# Project Rules

- Repo exclusivo para portfolio personal
- No introducir referencias a Kore, ERP o procesos internos de empresa
- Mantener stack simple: Vite + HTML/CSS/JS
- No romper contrato de GitHub Pages: `vite.config.js` con `base: '/portfolio/'`
- No eliminar Husky ni el hook de `npm run build` salvo instruccion explicita
- Cambios cerrados deben dejar `npm run build` pasando
